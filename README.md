# iShowcase

iShowcase is a front-end website made in React.js, CSS, and Framer Motion for animations. This website serves as a showcase for iPhones.

## Features

- **Modern Design:** Enjoy a sleek and modern design tailored for showcasing iPhones.
- **Interactive Animations:** Experience smooth animations powered by Framer Motion.
- **Responsive Layout:** Explore the website seamlessly across various devices and screen sizes.

## Getting Started

To get started with iShowcase, follow these simple steps:

1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/Nikhilv77/iShowcase.git
   ```

2. Navigate to the project directory.
   ```bash
   cd iShowcase
   ```

3. Install the necessary dependencies using npm.
   ```bash
   npm install
   ```

4. Run the development server.
   ```bash
   npm start
   ```

5. Access the website through your preferred web browser.
